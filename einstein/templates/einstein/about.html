I wrote this website because my students, whilst understanding the basic concepts of programming, constantly struggle with putting all of the different elements of a program together.

I find that many learners understand how to define a function or a method and,when working on this as a topic, completely understand what to do, how to do it, and how it all hangs together (values in, values back, etc.). However, when asked to write a program that uses functions or methods to actually do something, many students panic and have no idea where to start.

Also, when programming moves into more complex areas like reading multiple lists/arrays or csv files, error handling, or OOP, students can get frustrated, confused, and disheartened.

When frustrated, confused, or disheartened, students often turn to the internet for solutions. They search Google or Stack Overflow and often end up down a rabbit hole, more lost than when they started.

But no more (hopefully)!! This website provides learners with examples of perfectly working code in all of the major languages of today. From simple concepts to more challenging structures, all examples on this website work perfectly and can be easily copied into an IDE to be modifed as required.

I truly hope you find this website of value and that you will spread the word so that other learners may also have a simple, free collection of good clean working code.

Mike Maycock, Tutor - Software Developer Associate Apprenticeship Programme, GRETB, Galway, Ireland.
