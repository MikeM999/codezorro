Some quick notes on common JavaScript concepts...

Math.round(): var x = Math.round(0.9); This code rounds x to 1.
Math.pow(): var x = Math.pow(2,3); This code cubes 2, giving x the value of 8.
var x = Math.pow(2,2); This code squares 2, giving x the value of 4.
Math.sqrt(): var x = Math.sqrt(81); This code returns the square root of 81, giving x the value of 9.
Math.abs(): var x = Math.abs(-3); This code returns the absolute value of -3, giving x the value of 3.
Math.round(): var x = Math.round(1.4); This code rounds 1.4 to the nearest whole integer, giving x the value of 1.
var x = Math.round(3.7); This code rounds 3.7 to the nearest whole integer, giving x the value of 4.
var x = Math.round(-5.4); This code rounds -5.4 to the nearest whole integer, giving x the value of -5.
Math.ceil(): var x = Math.ceil(1.4); This code rounds 1.4 up to the nearest whole integer, giving x the value of 2.
var x = Math.ceil(-3.7); This code rounds -3.7 up to the nearest whole integer, giving x the value of -3.
Math.floor(): var x = Math.floor(4.2); This code rounds 4.2 down to the nearest whole integer, giving x the value of 4.
var x = Math.floor(-5.7); This code rounds -5.7 down to the nearest whole integer, giving x the value of -6.
Math.max(): var x = Math.max(4, 7); This code returns the highest value to x, giving x the value of 7.
Math.min(): var x = Math.min(4, 7); This code returns the lowest value to x, giving x the value of 4.
Math.random(): var x = Math.random(); This code generates a random number between 0 and 1 and assigns it to x.
Number.isInteger(): var x = Number.isInteger(12); This code returns a value of true to x because the supplied value (12) is an integer.
var x = Number.isInteger("12"); This code returns a value of false to x because the supplied value ("12") is not an integer.
var x = Number.isInteger(12.2); This code returns a value of false to x because the supplied value (12.2) is not an integer.
Number.isNaN(): var x = Number.isNaN(NaN); The Number.isNaN() method only returns true if the value being evaluated has a value of "NaN".
For example, if the result of a calculation is NaN, then Number.isNaN() will return true.
As such, the Number.isNaN() method should not be used to check is a variable has a number value or not.
